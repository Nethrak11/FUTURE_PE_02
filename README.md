# Prompt Engineering Task 2

## AI Content Marketing using UGC Ads

### Objective
The objective of this task is to generate user-generated content (UGC) style advertisement scripts using AI tools and prompt engineering techniques for social media marketing.

### Product
PowerFuel Protein Powder

### Tools Used
- ChatGPT
- Prompt Engineering
- GitHub for documentation

### Task Description
In this task, I used structured prompts to generate marketing content for a product. The AI generated multiple ad components including attention-grabbing hooks, short UGC-style advertisement scripts, call-to-action lines, and an Instagram caption.

The content is designed for short-form video platforms such as Instagram Reels and YouTube Shorts, focusing on authentic and engaging marketing communication.

### Project File
The complete documentation for this task is available in the uploaded PDF file in this repository.

### Learning Outcome
This task helped me understand how prompt engineering can be used to create effective marketing content. It also demonstrated how AI tools can assist businesses in producing engaging social media advertisements quickly.
